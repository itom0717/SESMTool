Space Engineers Same Mod Tool
====

このプログラムは Space Engineers のセーブファイルのMOD情報を別のセーブファイルへコピーします。  
新規World作成時にModを別のセーブデータと同じにするときに使います。


## 開発環境
 Microsoft Visual Studio Community 2015

## 必要ランタイム
 .NET Framework 4.5  

## 使い方

1.「Load SaveData」ボタンでMod一覧を取得したいセーブデータの Sandbox.sbc を読み込む。  
2. 現在のMOD一覧にModIDがリストされるので、「Mod一覧保存」ボタンで保存。  
3.「Load SaveData」ボタンでMod一覧を設定したいセーブデータの Sandbox.sbc を読み込む。  
4.「Mod一覧読込」ボタンで 2. で保存したMod一覧リストファイルを読み込む。  
5.「Save SaveData」ボタンで保存します。  

※変更前のファイルは Sandbox.bak で保存されます。
